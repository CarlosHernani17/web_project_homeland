# Tripleten web_project_homeland

Este é o projeto **Homeland** desenvolvido durante o Sprint 5 do programa de Web Development da TripleTen. O objetivo principal deste projeto foi aplicar os conceitos de **Design Responsivo (Adaptativo)** e organização de código utilizando a metodologia **BEM**, garantindo que a página web funcione perfeitamente em telas de Desktop, Tablet e Mobile.

## 🚀 Tecnologias Utilizadas

- **HTML5** (Estruturação semântica)
- **CSS3** (Flexbox, Media Queries com breakpoints de 544px e 1024px)
- **Metodologia BEM** (Block, Element, Modifier)
- **Fontes locais** conectadas via `@font-face`
